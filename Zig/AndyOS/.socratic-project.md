# Project: AndyOS

## Scope
- 512-byte MBR stage 1 bootloader in NASM, loaded at 0x7C00
- Prints "Hello" via INT 10h, AH=0x0E (teletype)
- Hangs after printing (CLI + HLT loop)
- Bootable under QEMU

## Prerequisites
- [/] Learn x86 real-mode memory model — ORG 0x7C00 (done), segments (pending)

- [ ] Learn building bootable images and running in QEMU
